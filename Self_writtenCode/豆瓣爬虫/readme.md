# 豆瓣书籍爬虫项目

🎯 **项目目标**  
本项目用于爬取豆瓣图书标签页下的书籍信息（如书名、评分、作者等），并将数据保存为 Excel 文件，方便用于后续的数据分析或推荐系统训练。

---

## 📦 功能说明

- 根据指定的豆瓣图书标签（如小说、科幻等）爬取所有分页书籍信息；
- 提取书籍标题、评分与基本信息（作者 / 出版社 / 出版时间 / 价格）；
- 自动保存至 Excel 文件，存放在桌面目录；
- 简单易用，方便数据采集与清洗。
